# Los Angeles Air Quality Analysis

## Overview
This project analyzes air quality trends in the Los Angeles region using publicly available environmental data. The goal is to explore pollution patterns and visualize trends over time.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Data cleaning and preprocessing
- Trend visualization
- Basic statistical analysis
